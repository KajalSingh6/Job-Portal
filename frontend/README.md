# JobHunt - Full Stack Job Portal

JobHunt is a modern full-stack job portal web portal web application where users can search and apply for jobs, and recruiters can post and manage job listings.

```

## Features

### User Features
-User Authentication (Singup/ Login)
- Browse & Search Jobs
- View Job Details
- Apply for Jobs
- Update Profile
- Update Resume (PDF)

### Admin / Recruiter Features
- Create & Manage Companies
- Post New Jobs
- Edit Existing Jobs
- View Applicants
- Manage Job Listings


---

## Tech Stack

## Frontend
- React (Vite)
- Redux Toolkit
- Tailwind CSS
- React Router DOM
- Axios


### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer (File Upload)
- Cloudinary (Resume Storage)
